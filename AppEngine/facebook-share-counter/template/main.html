About

This project was created by Luís Leão as a complement for iLikeDiesel campaign.

It uses Arduino, XBee and web's access to provide the user see how many people liked and interacted with a link in realtime.

The purpose is demonstrate the use of xbee radios and also realtime discover new devices in the network and provide web access controll of that devices.

You can download the source of this project on GitHub.
